# TradeLogMRTZ

Journal de trading personal con dashboard, estadísticas por sesión NY/Londres, capturas de pantalla y sistema de usuarios.

## Uso

Abrí la app en: **https://TU-USUARIO.github.io/tradelog-mrtz**

## Funcionalidades

- Login por usuario con contraseña
- Dashboard con winrate, P&L acumulado y gráfico
- Estadísticas separadas NY / Londres
- Adjuntar capturas de pantalla a cada trade
- Exportar / importar JSON como respaldo
- Instalable como app en Android e iPhone
